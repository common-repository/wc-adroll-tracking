Easily add AdRoll SmartPixel to your WooCommerce site and boost your sales.

== Description ==

AdRoll Retargeting is the only unified platform that helps you reach customers intelligently across devices on Facebook, Twitter, Apple iAd, and millions of websites.

Install AdRoll on your WooCommerce site now and start earning more.

Want more control over what info is sent to AdRoll?
Check our Pro version at http://pluginterritory.com/shop/woocommerce-adroll-tracking/


== Installation ==

1. Upload plugin files to your plugins folder, or install using WordPress' built-in Add New Plugin installer.
1. Activate the plugin.
1. Inform your AdRoll Advertisable ID and your Pixel ID.
1. Your WooCommerce store now has AdRoll tracking for the products.
